# CD 播放器功能教程

点击控制栏上的 **"💿" (CD)** 按钮，即可打开内置的音乐播放器。这里收录了 Project SEKAI 的丰富曲库，支持多种筛选和播放模式。

## 1. 播放控制

*   **基础控制**：播放/暂停、上一首、下一首。
*   **进度条**：拖动进度条可快速跳转。
*   **音量调节**：独立的音乐音量滑块，不影响背景视频的音量。
*   **播放模式**：
    *   🔀 **随机播放 (Shuffle)**：打乱当前列表的播放顺序。
    *   🔁 **单曲循环 (Repeat)**：循环播放当前歌曲。

## 2. 曲库浏览与筛选

播放器提供了强大的分类浏览功能：

### 侧边栏分类
*   **全部歌曲 (All Music)**：显示所有可用歌曲。
*   **我的收藏 (Favorites)**：显示您标记为“喜欢”❤️ 的歌曲。
*   **本地音乐 (Local Music)**：播放您导入的本地音频文件。
*   **播放列表 (Playlists)**：管理您的自定义歌单。
*   **按组合分类**：
    *   Leo/need
    *   MORE MORE JUMP!
    *   Vivid BAD SQUAD
    *   Wonderlands×Showtime
    *   25时，Nightcord见。
    *   VIRTUAL SINGER (虚拟歌手)
    *   Other (其他/混合组合)

### 搜索功能
在顶部的搜索框输入关键词，可以快速查找歌曲标题。

## 3. 演唱版本选择 (Vocal Preference)

Project SEKAI 的一大特色是同一首歌有多个演唱版本。本播放器完美支持这一功能。

### 自动偏好设置
点击播放器右上角的 **"⚙️" (设置)** 图标，可以设定您的**首选演唱版本**：
*   **Sekai Ver. (世界版)**：优先播放由游戏角色和虚拟歌手合唱的版本（默认）。
*   **Virtual Singer Ver. (虚拟歌手版)**：优先播放原版/虚拟歌手版本。
*   **Another Vocal Ver. (翻唱版)**：优先播放角色的个人翻唱或特定组合翻唱。

### 角色偏好 (Character Priority)
在设置中，您还可以选择您**最喜欢的角色**。
*   当一首歌有多个“Another Vocal”版本时，系统会优先选择您喜欢的角色所演唱的版本。
*   例如：如果您选择了“初音未来”，系统会优先播放有初音未来参与的翻唱版本。

### 手动切换
在播放界面，点击歌曲封面下方的按钮，可以随时手动切换当前歌曲的演唱版本。

## 4. 收藏与播放列表

### 收藏歌曲
*   在歌曲列表中，点击歌曲右侧的 **"❤️"** 图标即可收藏/取消收藏。
*   收藏的歌曲会出现在“我的收藏”分类中。

### 自定义播放列表
1.  点击侧边栏的 **"我的歌单"**。
2.  点击 **"新建歌单"** 创建新歌单。
3.  在歌曲列表中，点击 **"+"** 号，将歌曲加入指定歌单。
4.  您可以重命名或删除播放列表。

## 5. 本地音乐导入
*   点击侧边栏的 **"本地音乐"**。
*   点击 **"导入本地音乐文件..."** 按钮，选择您电脑上的 MP3/WAV/OGG 文件。
*   导入的音乐仅保存在浏览器缓存中，不会上传到服务器。

## 6. 数据来源
播放器会自动从云端获取最新的歌曲数据 (`musics.json`) 和演唱版本信息 (`musicVocals.json`)，确保曲库与游戏更新保持同步。
