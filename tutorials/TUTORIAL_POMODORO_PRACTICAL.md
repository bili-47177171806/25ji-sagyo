# 番茄钟实用教程

## 一、什么是番茄工作法

番茄工作法是由弗朗西斯科·西里洛于1992年创立的简单时间管理方法，通过将工作时间划分为小块，配合规律休息，提高工作和学习效率。

---

## 二、核心原则

| 原则 | 说明 |
|------|------|
| 番茄钟不可分割 | 一旦开始就不能暂停，中断则作废重来 |
| 大任务要拆分 | 超过6个番茄钟的任务需再拆分 |
| 小任务可合并 | 不到一个番茄钟的活动与其他活动合并 |
| 仅用于工作学习 | 不用于假期和休息期的活动 |

---

## 三、进阶使用指南

### 【一阶】基础版：25+5循环

**所需工具：** 计时器

**操作步骤：**
1. 选择一项任务
2. 开启25分钟倒计时，专注工作
3. 计时结束后休息5分钟
4. 每完成4个番茄钟，休息30分钟

**休息建议：** 喝水、简单活动，**不要做需要动脑的事**

---

### 【二阶】进阶版：增加今日待办

**所需工具：** 计时器 + 今日待办表 + 笔

**今日待办表格式：**

| 日期 | 时间 | 事项名称 | 番茄记录 | 干扰记录 |
|------|------|----------|----------|----------|
| 12/17 | 9:00 | 读《番茄工作法》 | ×××× | |

**操作步骤：**
1. 将大任务拆分成小任务写入待办表
2. 开始番茄钟，完成1个画一个×
3. 完成一项任务后用横线划掉

---

### 【三阶】高级版：处理干扰

**所需工具：** 计时器 + 今日待办表 + 活动清单 + 笔

**干扰类型及记录方式：**
- **内部干扰**（自己产生）：用斜线 **/** 记录
- **外部干扰**（他人造成）：用横线 **-** 记录
- （记录于今日待办表的“干扰记录”栏中）

**干扰处理流程：**

```
遇到干扰 → 判断是否今天必须做
    ↓是                    ↓否
记入"计划外紧急事件"    记入"活动清单"
    ↓                      ↓
记录干扰符号 → 立即返回番茄钟
```

> 注：用U表示不是今天要完成的计划外事件。如果这个事件需要在某天完成，那么就在时间栏标注U（时间），如U（12月21日）。


**处理技巧：**
- **提前屏蔽：** 手机静音、关闭通知
- **延后处理：** 2分钟内能解决则立即处理，否则推迟

---

### 【四阶】完整版：记录分析

**所需工具：** 计时器 + 今日待办表 + 活动清单 + 记录表 + 笔

**完整流程（5阶段）：**

| 阶段 | 内容 |
|------|------|
| 计划 | 预估每项任务所需番茄数（用□表示） |
| 追踪 | 标记完成情况（在小方框□中画一个叉×） |
| 记录 | 记录实际番茄数和干扰次数 |
| 分析 | 对比预估与实际，计算误差 |
| 可视化 | 汇总数据，制作图表分析趋势 |

> 例：在完成了2个番茄钟之后，发现还没完成任务的一半，那么3个番茄钟可能就不能完成这个事项，所以可以进行第二次预估，使用小圆圈○来表示。比如预估还需要2个番茄钟，那么就在番茄记录中增加2个小圆圈○。

**预估技巧：**
- 第一次预估用 **□** 表示
- 执行中需二次预估用 **○** 表示
- 目标：逐步提高第一次预估准确率

## 五、常见问题

**Q：25分钟太长/太短怎么办？**
A：可根据个人情况调整，找到适合自己的节奏

**Q：刚开始坚持不了4个循环？**
A：正常现象，从1-2个开始逐步增加

**Q：休息时想继续工作？**
A：坚持休息，这是恢复精力的关键
